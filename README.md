# Performing Exploratory Data Analysis(EDA) on Students performance Data

![BABy_Imge](https://www.icds.psu.edu/wp-content/uploads/2022/06/classroom-web.jpg)

This dataset contains information on the marks obtained by students in various subjects. The goal of this dataset is to investigate the impact of factors such as parental background and test preparation on student performance.

## About the Data

The data was generated using [Royce Kimmons' data generator](http://roycekimmons.com/tools/generated_data/exams). The data contains 1,000 observations and includes the following variables:

- gender: Male or Female
- race/ethnicity: Group A, Group B, Group C, Group D, or Group E
- parental level of education: Some high school, high school degree, some college, associate's degree, bachelor's degree, or master's degree
- lunch: Standard or Free/reduced
- test preparation course: Completed or None
- math score: Score (out of 100) on math exam
- reading score: Score (out of 100) on reading exam
- writing score: Score (out of 100) on writing exam

## Objective

The objective of this dataset is to perform Exploratory Data Analysis (EDA) to investigate the following questions:

- How are the scores distributed across the different subjects?
- What is the relationship between test preparation and scores?
- Is there a difference in scores based on parental level of education or race/ethnicity?
- Are there any other interesting trends or relationships in the data that can be explored?
